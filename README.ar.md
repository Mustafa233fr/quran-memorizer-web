# موقع اختبار حفظ القرآن

تطبيق ويب بسيط يساعد على حفظ القرآن الكريم واختبار المعرفة بالآيات.

## المميزات

* 📖 عرض آية عشوائية من القرآن الكريم
* 🎯 عدة أوضاع للاختبار:

  * آية عشوائية
  * تحديد نطاق صفحات
  * اختيار سورة معينة
  * وضع الجزء
* 🔊 تشغيل صوت الآيات بعدة قرّاء
* 🌙 التبديل بين الوضع الليلي والنهاري

## التقنيات المستخدمة

* HTML / CSS / JavaScript
* واجهة برمجة تطبيقات القرآن: https://alquran.cloud/api
* الصوت من: everyayah.com

## طريقة الاستخدام

1. افتح الموقع
2. اختر وضع الاختبار
3. أجب على الأسئلة
4. تحقق من النتائج
