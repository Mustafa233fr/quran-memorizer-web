# Quran Memorizer Web

A simple web app to help memorize and test knowledge of the Quran.

## Features

* 📖 Displays a random ayah from the Quran
* 🎯 Multiple quiz modes:

  * Random ayah
  * Page range
  * Specific surah
  * Juz mode
* 🔊 Audio playback from different reciters
* 🌙 Dark / Light mode toggle

## 🔗 Live Website

https://mustafa233fr.github.io/quran-memorizer-web/


## Tech Stack

* HTML / CSS / JavaScript
* [AlQuran Cloud API](https://alquran.cloud/api)
* Audio from everyayah.com

## How to Use

1. Open the website
2. Choose a mode
3. Answer the questions
4. Check your results
